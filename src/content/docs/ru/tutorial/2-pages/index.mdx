## Куда мы направляемся?

На этом шаге мы сделаем многе:
- Создадите свои первые Astro страницы с синтаксисом `.astro`
- Добавите блог-посты с помощью Markdown (`.md`) файлов
- Оформите индивидуальную страницу с помощью `<style>`
- Примените глобальные стили на страницах

 По пути вы узнаете, **как два раздела файла `.astro`** работают вместе для создания страницы, и как использовать переменные и условное рендеринг на ваших страницах.
